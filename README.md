# Projeto-datasets-aninhados-JSON-Alura---transforma-o-e-manipula-o-de-dados-Google-Colab-Python-
Análise de dados de um evento de 5 dias em um e-commerce para identificar o maior comprador da semana (premiado) e mapear clientes que mais gastaram. O projeto gera insights estratégicos para fidelização e atração de novos consumidores.
